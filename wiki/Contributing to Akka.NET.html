Akka.NET exposes the Actor model to .NET in a way that has never been done before, and it's a large project that encompasses a big range of capabilities. Therefore, contributions are welcome and encouraged.

How to Send Code Contributions to Akka.NET

If you want to contribute code to Akka.NET, here's how:

  1. First read the Developer Guidelines
  2. Fork this repository.
  3. Please squash your commits into a single commit, if possible - this will make it easier for everyone else to review. And be sure to include a description of your changes along with any related issues.
  4. Push changes to your local fork, and when you're ready to have them reviewed by an AkkaDotNet team member, send a pull request to the dev branch on this repository.
